## Como usar

```bash
# Clonar o repositório
$ git clone https://github.com/MatheusINFO/WhatsTalk.git

# Ir dentro do repositório clonado
$ cd WhatsTalk

# Instalar as dependências
$ yarn install

# Rodar o app
$ yarn start
```
